; Satellite-specific behavior tests.

;===============================================================
; Satellite needs attention only without target.
;===============================================================
GIVEN game map
  V#
GIVEN V is waiting for input.

THEN the game is waiting for input.

;===============================================================
; Satellite with target does not need attention.
;===============================================================
GIVEN game map
  V###%
GIVEN V's target is %.
GIVEN player units V.

WHEN player items are processed.

THEN the game is not waiting for input.

;===============================================================
; Satellite spawns with turns-remaining 50.
;===============================================================
GIVEN game map
  V#

THEN V has 50 turns remaining.

; Satellite moves 10 cells per round.

GIVEN game map
  V~~~~~~~~~=
GIVEN V is waiting for input.
